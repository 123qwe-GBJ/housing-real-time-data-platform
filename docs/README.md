# 加州房屋市场实时数据分析与可视化大屏系统
## 一、项目简介
本项目基于大数据流式处理技术，模拟加州房屋业务数据的实时采集、清洗存储、多维统计分析，最终通过前后端整合的可视化大屏，完成房源收入、房龄、户型、经纬度房价、房价收入比等多维度数据可视化展示，同时提供高价房源榜单、房产投资潜力区域推荐功能，可辅助房产从业者完成市场数据分析与投资决策。

## 二、技术栈
### 后端大数据
- 数据源：Java + Kafka 实现流式模拟数据生产
- 实时计算：Scala + Spark Streaming 流式数据消费、脏数据过滤、批量入库
- 数据存储：MySQL（ODS原始层 + DWS多维汇总层）
- 数据访问：SpringBoot + MyBatis 开发REST接口

### 前端可视化
- 页面框架：Thymeleaf + Vue + jQuery
- 可视化图表：ECharts（柱状图、饼图、折线图、地理热力图等）

### 开发环境
- JDK8、Scala 2.12、Kafka 2.4、Spark 2.4、MySQL 8.0、Maven

## 三、整体数据流转流程
1. **数据生产**：`data-generator` 模块定时循环模拟生成加州房屋结构化JSON数据，发送至Kafka的`housing-topic`主题。
2. **实时采集清洗**：`spark-streaming-job` 模块通过Direct方式消费Kafka消息，解析JSON数据，过滤异常脏数据，使用自建连接池批量将有效数据写入MySQL原始明细表。
3. **离线多维统计**：执行SQL脚本对原始房源数据进行多维度聚合统计，生成收入区间、房龄、户型、经纬度等维度的汇总分析表。
4. **接口数据服务**：`springboot-api` 模块封装各类统计、明细、榜单查询接口，为前端大屏提供数据支撑。
5. **可视化展示**：前端页面通过ECharts拉取后端接口数据，完成多维度图表可视化、房源登录权限校验、房产投资推荐展示。

## 四、项目仓库目录结构
```
├── sql                 # 数据库建表语句、ODS层原始表、DWS层多维统计分析SQL脚本
├── data-generator      # Kafka模拟房源数据生产者：Java实现，循环生成JSON格式业务数据并发送到Kafka集群
├── spark-streaming-job # Spark Streaming实时流处理模块：消费Kafka数据、清洗过滤、批量写入MySQL
├── springboot-api      # SpringBoot后端接口服务，整合ECharts可视化大屏前端页面，提供所有业务查询接口
├── docs                # 项目架构流程图、数据流转图、数据库设计文档、部署文档、课程设计报告
```

## 五、项目部署运行步骤
### 1. 环境准备
1. 启动Linux虚拟机，部署并启动ZooKeeper、Kafka集群服务，创建`housing-topic`主题。
2. 启动MySQL服务，执行`sql`目录下所有建表语句、初始化统计SQL脚本。

### 2. 启动数据生产者
运行`data-generator`下`HousingDataProducer`主类，开始定时生成模拟房源数据并发送至Kafka。

### 3. 启动Spark实时任务
运行`spark-streaming-job`下`HousingStreamToMySQL`流式任务，设置批处理间隔10秒，实时消费Kafka消息并入库。

### 4. 启动后端可视化服务
运行`springboot-api`项目的`AppStart`启动类，访问项目登录地址，登录后即可进入房产数据分析可视化大屏，查看多维度统计图表与投资推荐数据。

## 六、项目核心功能
1. 实时房源数据采集、异常数据过滤、批量稳定入库
2. 房屋收入、房龄、户型、地理位置多维度聚合分析
3. 高价房源TOP10榜单、房产市场综合评分分析
4. 加州区域房价热力图可视化展示
5. 房产投资潜力区域智能推荐
6. 系统登录权限校验、数据大屏多类型图表展示
