# 项目核心数据集
## 数据集说明
本数据集为加州房屋价格清洗后的结构化数据集，是本项目所有数据分析、实时流处理、可视化展示的核心业务数据源，已完成缺失值填充、异常值过滤、字段标准化清洗。

## 数据集基础信息
- 文件名：california_housing_prices_cleaned.csv
- 数据规模：约2万条加州房屋样本数据
- 数据来源：加州房屋公开数据集，已完成业务化清洗适配

## 核心字段说明
| 字段名 | 字段含义 | 数据类型 |
|--------|----------|----------|
| med_inc | 区域收入中位数 | 浮点型 |
| house_age | 房屋平均房龄 | 浮点型 |
| ave_rooms | 平均房间数 | 浮点型 |
| ave_bedrms | 平均卧室数 | 浮点型 |
| population | 区域人口数 | 整型 |
| ave_occup | 平均入住人数 | 浮点型 |
| latitude | 房屋纬度 | 浮点型 |
| longitude | 房屋经度 | 浮点型 |
| med_house_val | 房屋价值中位数 | 浮点型 |
| price_per_room | 单房间平均价格 | 浮点型 |
| income_to_price_ratio | 收入房价比 | 浮点型 |
